# **App Name**: Habit Planner

## Core Features:

- Daily Logging: Users log their meals and habits each day.
- Suggested Replacements: AI powered tool that provides alternative meals based on health goals.
- Goal Setting: Define daily/weekly goals related to diet.
- Progress Tracking: Visualize progress via charts.
- Personalized Tips: AI tool gives users custom advice, tailored to goals and food logs.

## Style Guidelines:

- Primary color: Soft blue (#64B5F6) to evoke calm and health.
- Background color: Very light blue (#E3F2FD), close in hue to the primary.
- Accent color: Pale violet (#7986CB) for buttons and highlights.
- Body and headline font: 'PT Sans', a versatile and friendly sans-serif for readability.
- Clean, modern icons representing different food groups and habits.
- Simple, card-based layout to focus on daily entries and progress.
- Gentle animations on goal completion and data updates.
