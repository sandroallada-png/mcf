# Règles du Projet My Cook Flex

## Design & UI
- **Carrousels** : NE JAMAIS écrire de texte (h3, p, div) par-dessus les images des carrousels. Les images doivent rester vierges de toute superposition de texte hardcodé dans le code.
- **Identité AI** : Toujours utiliser "notre outil" ou "votre outil" au lieu de "IA" ou "Assistant IA" pour désigner les fonctionnalités intelligentes.

## Langue
- Toujours répondre à l'utilisateur en français.
